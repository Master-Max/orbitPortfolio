What is a Website?

Ask different people and you will get different answers. A bunch of ones and zeros, a text document that a browser can interpret, "What do you mean? It's just a website, like a google or the facebook or whatever."

The truth is a website can be whatever one can imagine, with enough lines of code you can make almost anything on the modern web. With the flexibility modern browsers offer, full scale applications can be developed with relative ease. These rich web experiences are unlike the simple blogs and catalogues that I grew up with. Applications like Skype, MSWord, and Photoshop can now be accessed through the web instantly, no installation required.

These tools are available to everyone, but you still need someone with the know how and dedication to implement them. That's where I come in.
